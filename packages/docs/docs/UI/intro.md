# Introduction

Reusable complex DHIS2 UI components for react applications

## Getting Started

The package is available in the npm. It can be installed through *yarn*, or *npm*

```bash
 yarn add @hisptz/dhis2-ui
```

or

```bash
 npm install @hisptz/dhis2-ui
```
